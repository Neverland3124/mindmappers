Click and move instantly on mindmap to drag.

Click and wait a moment to select multiple nodes.

Double click on mindmap to create a new node.

Backspace / Delete on mindmap to delete node(s).

Tab / Enter to create a new child/sibling for the selected node.

W / S key to go to parent/child node.

Ctrl + C / X / V on mindmap to copy/cut/paste selected node(s).

Ctrl + Z / Y on mindmap to undo/redo action.

Ctrl + Scroll on mindmap to zoom in/out.

Shift + Enter when editing node to start a new line.
